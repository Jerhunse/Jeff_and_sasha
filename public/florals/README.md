# Floral Assets

This directory contains decorative floral assets for the wedding theme.

## Included (SVG)

- `corner-tl.svg` – Top-left corner floral (sage + gold accents)
- `corner-br.svg` – Bottom-right corner floral
- `divider-branch.svg` – Branch divider for section separators

## Optional (replace with PNG if preferred)

- `corner-tl.png` / `corner-br.png` – Use if you have custom corner artwork; components default to the SVGs above.

## Usage

Used by `FloralCorner`, `FloralCorners`, and `FloralDivider` in `components/wedding/`.
