# 💒 Wedding Platform - Complete & Ready

A **spec-compliant, production-ready wedding platform** with advanced features for managing every aspect of your wedding.

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Set up environment
cp .env.example .env
# Edit .env with your credentials

# 3. Run migration
./scripts/run-refactor-migration.sh

# 4. Start development
npm run dev

# 5. Access admin
open http://localhost:3000/admin/overview
```

## ✨ Features

### For Couples (Admin)
- 📧 **Campaign Management** - Send Save-the-Dates, Invitations, Reminders
- 🎨 **Visual Page Builder** - 8 section types, live preview
- 📸 **Media Library** - Upload, tag, and manage images
- 🎭 **Theme Customizer** - Colors, fonts, style with presets
- 👥 **Guest CRM** - Import, tag, manage households
- ❓ **Dynamic RSVP Questions** - 12 question types, conditional logic
- 💺 **Seating Charts** - Drag-drop tables, conflict detection
- 📨 **Messaging** - Targeted messages with segmentation
- 📊 **Analytics** - Real-time stats, charts, trends
- 🔍 **Dedupe Tools** - Fuzzy matching, easy merging
- 🏨 **Hotel Blocks** - Manage accommodations
- 📥 **Import from Google** - OAuth contacts integration

### For Guests
- 📱 **Responsive RSVP** - Beautiful forms, any device
- 🎫 **Guest Portal** - Magic link access
- ✏️ **Update Info** - Edit contact details
- 📋 **View History** - See RSVP and invitations
- 💺 **Seating Info** - Know your table assignment
